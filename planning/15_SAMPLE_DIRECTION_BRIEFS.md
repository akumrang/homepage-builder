# 샘플 방향 3종 브리프

Status: Draft  
Project: homepage  
Last Updated: 2026-06-28

---

## 1. 목적

이 문서는 향후 완성형 샘플 갤러리를 만들기 전에 필요한 샘플 방향 3종의 브리프를 정의한다.

현재 목표는 구현이 아니다. `trust-basic-v1`, `exam-focus-v1`, `small-group-v1` 후보가 각각 어떤 학원에 맞고, 어떤 자료가 필요하며, 어떤 문구를 금지해야 하는지 묵산 내부 제작 기준으로 고정하는 것이다.

샘플 방향은 고객이 자유롭게 홈페이지를 조립하는 선택지가 아니다. 고객은 선호 분위기를 고르고, 묵산은 학원 자료와 운영 가능 범위 안에서 제작 방향을 확정한다.

---

## 2. 공통 원칙

모든 샘플 방향은 다음 원칙을 공유한다.

- 실제 개인정보를 사용하지 않는다.
- 허위 합격 실적, 허위 후기, 허위 강사 경력을 만들지 않는다.
- 학부모가 신뢰할 수 있는 첫인상을 우선한다.
- 상담 신청 동선은 항상 명확해야 한다.
- 학원 사진이 없어도 깨지지 않아야 한다.
- 고객별 별도 코드베이스를 전제로 하지 않는다.
- 자유형 드래그앤드롭 편집기처럼 보이면 안 된다.
- 선택값은 자동 게시 명령이 아니라 묵산 내부 제작 입력값이다.

공통 필수 섹션:

| 섹션 | 목적 |
|---|---|
| 메인 | 학원명, 대표 문구, 상담 CTA |
| 소개 | 수업 철학, 운영 방식, 관리 방식 |
| 강사진 | 확인 가능한 역할과 수업 초점 |
| 커리큘럼 | 학년/과목별 수업 흐름 |
| 공지 | 상담, 개강, 시험 대비 등 최신 안내 |
| 오시는 길 | 주소, 운영 시간, 연락처 |
| 상담 신청 | 보호자 문의 접수 |

공통 금지:

- "100% 합격", "무조건 등급 상승", "지역 1위"처럼 검증 불가능한 표현
- 실제 후기처럼 보이는 가짜 문장
- 확인되지 않은 대입, 학교, 직장, 수상 경력
- 학생 성적, 학교, 이름 등 개인정보 노출
- 특정 외부 사이트를 그대로 복제한다는 표현

## 3. 방향 A: `trust-basic-v1`

### 성격

신뢰형 기본 방향이다.

이미 MVP에서 `sample-korean-academy`에 적용된 기준 방향이며, 과장 없는 안정감과 정보 전달을 우선한다.

### 적합한 학원

- 국어, 영어, 보습, 사회 등 중소형 단과 학원
- 초중등부터 고1 기본기까지 다루는 학원
- 강한 입시 실적보다 수업 흐름과 관리 방식을 설명하고 싶은 학원
- 사진이나 브랜드 자산이 많지 않아도 안정적인 홈페이지가 필요한 학원

### 맞지 않는 경우

- 화려한 브랜드 이미지나 대형 학원 인상을 강하게 원할 때
- 합격 실적, 입시 결과, 대형 설명회 중심으로 설득해야 할 때
- 시설 사진과 브랜드 자산이 많아 이를 크게 보여주고 싶을 때

### 첫 화면 방향

| 항목 | 기준 |
|---|---|
| 첫 인상 | 차분함, 신뢰, 안정 |
| 대표 문구 | 학습 습관과 수업 관리 중심 |
| CTA | 상담 신청, 수업 흐름 확인 |
| 색상 | 저채도 남색, 녹색, 회색 계열 선호. 과한 원색 금지 |
| 이미지 | 실제 수업 공간 또는 차분한 학습 이미지. 없으면 생성/샘플 이미지 가능 |

### 필수 자료

- 학원명과 로고 또는 텍스트 로고 승인
- 대표 문구 1개
- 대상 학년과 과목
- 핵심 강점 3개
- 수업 철학, 운영 방식, 관리 방식
- 강사진 1~3명 소개
- 커리큘럼 2~4개
- 공지 1~3개
- 주소, 전화번호, 운영 시간, 교통/주차 안내

### 가능한 조정

- 대표 색상 선호
- hero 이미지 또는 대체 이미지 방향
- 강점 3개의 순서
- 상담 문구의 톤
- 강사진 소개를 이름 중심 또는 역할 중심으로 표현

### 금지 문구

- 빠른 성과를 보장하는 문구
- 실제처럼 보이는 가짜 후기
- 확인되지 않은 강사 경력
- 특정 학교 내신 적중을 암시하되 근거가 없는 문구

### 운영 난이도

낮음.

현재 MVP 구조와 가장 잘 맞는다. 사진 부족, 로고 부재, 짧은 강사 소개에도 대응하기 쉽다. 첫 파일럿 고객에게 가장 안전한 기본 방향이다.

### 갤러리 카드 문안 초안

```text
신뢰형 기본
과장 없이 학원의 수업 흐름과 관리 방식을 차분하게 보여주는 기본 방향입니다.
자료가 많지 않은 중소형 단과 학원에도 안정적으로 적용할 수 있습니다.
```

## 4. 방향 B: `exam-focus-v1`

### 성격

내신/입시 관리형 방향이다.

시험 대비, 학년별 커리큘럼, 자료 관리, 학습 점검 체계를 선명하게 보여주는 데 집중한다.

### 적합한 학원

- 고등 내신 또는 수능 대비를 중심으로 운영하는 학원
- 중3부터 고등부 전환 관리를 강조하는 학원
- 학교별 시험 범위, 기출, 오답 관리 등 체계가 있는 학원
- 상담 시 현재 성적보다 학습 계획과 관리 루틴을 설명해야 하는 학원

### 맞지 않는 경우

- 입시 실적이나 점수 향상 근거를 제공할 수 없는 상태에서 결과 중심 홍보를 원할 때
- 초등 저학년, 독서, 기초 문해력 중심 학원
- 따뜻하고 부드러운 첫인상을 우선해야 하는 학원

### 첫 화면 방향

| 항목 | 기준 |
|---|---|
| 첫 인상 | 전문성, 체계, 시험 대비 |
| 대표 문구 | 내신/수능 대비 구조와 관리 루틴 중심 |
| CTA | 상담 신청, 학년별 대비 과정 확인 |
| 색상 | 선명한 navy, charcoal, restrained accent. 과한 경쟁/자극 색상 금지 |
| 이미지 | 교재, 학습 계획표, 강의실, 시험 대비 자료. 실제 학생 개인정보 노출 금지 |

### 필수 자료

- 대상 학년과 시험 대비 범위
- 내신 대비 방식
- 수능 또는 고등 대비 방식
- 학교별/학년별 관리 가능 범위
- 커리큘럼 3~5개
- 수업 자료 관리 방식
- 과제/오답/피드백 루틴
- 확인 가능한 입시 실적이 있다면 근거 자료
- 실적이 없다면 실적 섹션을 만들지 않는다는 동의

### 가능한 조정

- 내신 중심 또는 수능 중심 강조 비율
- 학년별 과정 노출 순서
- 시험 대비 공지 강조 여부
- 상담 CTA의 톤
- 성과보다 관리 시스템을 더 앞세우는 구성

### 금지 문구

- "무조건 1등급", "전원 성적 상승", "합격 보장"
- 근거 없는 학교별 적중률
- 실제 학생 성적표, 이름, 학교가 보이는 이미지
- 합격생 인터뷰처럼 보이는 허위 후기
- 강사의 입시 경력을 확인 없이 권위 표현으로 쓰는 것

### 운영 난이도

중간.

자료 검수가 중요하다. 고객이 실적 중심 문구를 요구할 가능성이 높으므로, 사실 확인 자료가 없으면 성과 표현을 빼고 관리 체계 중심으로 설계해야 한다.

### 갤러리 카드 문안 초안

```text
내신/입시 관리형
시험 대비 과정, 학년별 커리큘럼, 오답 관리 흐름을 선명하게 보여주는 방향입니다.
성과 문구는 확인 가능한 자료가 있을 때만 사용합니다.
```

## 5. 방향 C: `small-group-v1`

### 성격

소수정예 관리형 방향이다.

대형 학원보다 작은 반, 밀착 피드백, 학부모 소통, 과제 점검을 강점으로 삼는 학원에 맞춘다.

### 적합한 학원

- 반별 정원이 작고 관리 흐름이 뚜렷한 학원
- 과제, 오답, 학습 습관, 상담 기록을 꾸준히 보는 학원
- 학부모와의 소통을 중요하게 운영하는 학원
- 중하위권 보완, 기초 회복, 학생별 피드백을 강조하는 학원

### 맞지 않는 경우

- 대형 입시 브랜드처럼 보이기를 원하는 학원
- 많은 강사진과 대규모 시스템을 강조해야 하는 학원
- 실제 관리 방식이 정리되어 있지 않은데 밀착 관리를 주장하려는 학원

### 첫 화면 방향

| 항목 | 기준 |
|---|---|
| 첫 인상 | 따뜻함, 꼼꼼함, 가까운 관리 |
| 대표 문구 | 작은 반, 꾸준한 점검, 학부모 소통 |
| CTA | 상담 신청, 관리 방식 확인 |
| 색상 | 부드러운 green, warm gray, muted blue. 과한 프리미엄/고급감 금지 |
| 이미지 | 상담실, 교재 점검, 소규모 수업 분위기. 실제 학생 식별 금지 |

### 필수 자료

- 반별 정원 또는 운영 가능한 정원 기준
- 과제 점검 방식
- 오답 또는 피드백 방식
- 학부모 소통 방식
- 학생 변화 관찰 기준
- 강사진 역할과 관리 초점
- 커리큘럼 2~4개
- 상담 가능 시간과 상담 전 확인할 항목

### 가능한 조정

- 따뜻한 톤 또는 단정한 톤 선택
- 학부모 소통 강조 정도
- 수업 후 점검, 과제 관리, 오답 관리 중 우선순위
- 강사 이름 노출 여부
- 사진이 부족할 때 텍스트 중심 구성 강화

### 금지 문구

- "일대일처럼 완벽 관리"처럼 운영 현실을 넘는 표현
- 확인되지 않은 학생 변화 사례
- 학부모 후기 창작
- 고급/프리미엄 이미지만 앞세워 실제 관리 방식이 비는 구성
- 강사의 세부 개인정보나 확인되지 않은 경력 강조

### 운영 난이도

중간.

고객이 실제 관리 루틴을 구체적으로 제공해야 설득력이 생긴다. 자료가 부족하면 감성 문구만 남아 싸구려 홍보 페이지처럼 보일 위험이 있다.

### 갤러리 카드 문안 초안

```text
소수정예 관리형
작은 반, 과제 점검, 학부모 소통처럼 가까운 관리 방식을 보여주는 방향입니다.
실제 운영 루틴이 있을 때 가장 설득력이 좋습니다.
```

## 6. 방향별 비교

| 항목 | `trust-basic-v1` | `exam-focus-v1` | `small-group-v1` |
|---|---|---|---|
| 핵심 가치 | 안정감과 신뢰 | 시험 대비 체계 | 밀착 관리 |
| 첫 파일럿 적합도 | 높음 | 중간 | 중간 |
| 자료 요구량 | 낮음 | 높음 | 중간 |
| 검수 난이도 | 낮음 | 높음 | 중간 |
| 허위/과장 위험 | 낮음 | 높음 | 중간 |
| 사진 의존도 | 낮음 | 중간 | 중간 |
| 현재 구현 상태 | 구현됨 | 브리프만 존재 | 브리프만 존재 |

초기 실제 제작 우선순위는 다음과 같다.

1. `trust-basic-v1`: 이미 구현된 기준 방향으로 첫 파일럿에 사용 가능
2. `small-group-v1`: 소규모 학원 고객이 많을 경우 다음 후보
3. `exam-focus-v1`: 수요는 크지만 실적/성과 문구 검수 리스크가 높으므로 자료 검수 체계 이후 권장

## 7. 갤러리 구현 전 남은 결정

샘플 방향 브리프는 갤러리 구현의 충분조건이 아니다.

다음 항목이 추가로 필요하다.

- 각 방향의 실제 샘플 문구 세트
- 각 방향의 최소 샘플 이미지 또는 placeholder 정책
- 샘플 카드 UI 문안
- 고객 선택값을 기록할 자료 수집 양식 항목
- 선택 후 묵산 내부 제작자가 판단할 체크리스트
- 고객에게 보여줄 기대 관리 문구
- 고객이 선택한 방향과 실제 제작 결과가 달라질 수 있는 사유 안내

다음 1순위 추천 작업은 이 변경분을 커밋하고 원격 저장소에 푸시하는 것이다.

그 작업도 구현이 아니다. 고객에게 보여줄 샘플 카드 제목, 설명, 적합/부적합 안내, 선택 전 고지 문구를 문서로 확정한다.
