# 홈페이지 MVP 구현 범위

Status: Draft  
Project: homepage  
Last Updated: 2026-06-24

---

## 1. 목적

이 문서는 묵산 홈페이지 프로젝트의 첫 코드 구현 범위를 확정한다.

목표는 모든 기능을 한 번에 구현하는 것이 아니라, 홈페이지 프로젝트의 핵심 구조가 실제로 작동하는 최소 단위를 만드는 것이다.

---

## 2. MVP 구현 목표

첫 구현 목표는 다음이다.

> 샘플 학원 데이터를 기반으로 묵산 표준 템플릿 1종을 렌더링하고, 상담 문의를 저장할 수 있는 최소 홈페이지 시스템을 만든다.

즉, 초기 MVP는 다음을 증명해야 한다.

- 하나의 공통 코드로 학원별 홈페이지를 렌더링할 수 있다.
- 템플릿과 학원별 콘텐츠 데이터를 분리할 수 있다.
- 모바일에서 신뢰감 있는 학원 홈페이지가 보인다.
- 상담 문의가 접수된다.
- 내부 제작·검수·게시 흐름으로 확장 가능한 구조다.

---

## 3. 포함 범위

MVP에 포함할 기능은 다음이다.

### 3-1. 프로젝트 기본 구조

- frontend 앱
- backend 앱
- TypeScript 설정
- 기본 실행 스크립트
- 개발 포트 분리

권장 포트:

| 영역 | 포트 |
|---|---:|
| frontend | 5175 |
| backend | 4200 |

---

### 3-2. 샘플 학원 데이터

초기에는 실제 DB 없이 샘플 JSON 또는 메모리 데이터를 사용할 수 있다.

샘플 데이터는 다음을 포함한다.

- 학원명
- slug
- 대표 문구
- 대표 과목
- 대상 학년
- 핵심 강점
- 학원 소개
- 강사진
- 커리큘럼
- 시간표 또는 수업 안내
- 공지사항
- 주소
- 전화번호
- 상담 안내

실제 개인정보는 사용하지 않는다.

---

### 3-3. 공개 홈페이지 렌더링

다음 URL 구조를 우선 검토한다.

```text
/h/:slug
예:

/h/sample-korean-academy

초기에는 slug 기준으로 샘플 학원 홈페이지를 렌더링한다.

3-4. 템플릿 1종

초기 템플릿은 다음으로 한다.

trust-basic-v1

목표:

모바일 우선
깔끔한 학원 첫인상
상담 신청 버튼 명확
메인, 소개, 강사진, 커리큘럼, 공지, 오시는 길 표시
과장 광고 느낌 지양
3-5. 상담 문의 접수

상담 문의 폼을 구현한다.

입력 항목:

보호자 이름
연락처
학생 학년
관심 과목
문의 내용
개인정보 수집 동의

초기 저장 방식:

1단계: backend 메모리 저장 또는 JSON 로그
2단계: Prisma DB 저장

MVP에서는 최소한 POST 요청이 backend로 전달되고 저장되는 것을 확인한다.

3-6. 내부 제작 화면 최소판

초기 내부 화면은 아주 단순하게 둔다.

포함 후보:

샘플 학원 목록
홈페이지 미리보기 링크
상담 문의 목록
게시 상태 표시

이 화면은 고객용 편집기가 아니다.

묵산 내부 제작자용 화면이다.

5. 기술 스택 초안

academy 프로젝트와 장기 통합을 고려하여 다음 구조를 우선 검토한다.

영역	선택
Frontend	Vite + React + TypeScript
Backend	Express + TypeScript
Database	PostgreSQL + Prisma
Styling	CSS 또는 CSS Modules
AI	후순위, API 경계만 준비
Hosting	후순위

초기 MVP는 DB 없이 시작할 수 있으나, 구조는 Prisma/PostgreSQL로 확장 가능하게 둔다.

7. 첫 구현 완료 기준

첫 구현은 다음을 만족하면 완료로 본다.

frontend 개발 서버가 실행된다.
backend 개발 서버가 실행된다.
/h/sample-korean-academy에서 샘플 홈페이지가 보인다.
모바일 폭에서도 화면이 깨지지 않는다.
상담 문의 폼을 제출할 수 있다.
backend가 상담 문의를 저장하거나 로그로 확인한다.
내부 제작 화면에서 문의 목록 또는 샘플 홈페이지 상태를 볼 수 있다.
README에 실행 방법이 기록된다.
Git 커밋이 남아 있다.
8. 판단 원칙

MVP는 작아야 한다.

그러나 품질이 낮아서는 안 된다.

기능은 좁게, 화면은 실제 상품처럼 만든다.

9. 한 줄 정의

홈페이지 MVP는 샘플 학원 데이터를 기반으로 고품질 기본 홈페이지 1종을 렌더링하고 상담 문의를 받을 수 있는 최소 실사용 구조다.
