# 샘플 카드 문안과 고객 기대 관리 문구 초안

Status: Draft  
Project: homepage  
Last Updated: 2026-06-28

---

## 1. 목적

이 문서는 향후 완성형 샘플 갤러리 또는 상담용 샘플 자료에서 고객에게 보여줄 카드 문안과 기대 관리 문구를 정의한다.

현재 목표는 구현이 아니다. 고객이 샘플을 보며 자신의 학원에 맞는 방향을 이해하되, 자유형 홈페이지 빌더나 즉시 자동 게시 기능으로 오해하지 않도록 문구 기준을 고정한다.

---

## 2. 전체 안내 문구

샘플 목록 상단에는 다음 문구를 둔다.

```text
아래 샘플은 우리 학원에 어울리는 홈페이지 방향을 함께 고르기 위한 참고 자료입니다.
선택하신 방향은 최종 디자인 확정이 아니라 제작을 시작하기 위한 기준점입니다.
최종 홈페이지는 학원 자료 확인, 묵산 제작, 내부 검수, 고객 확인을 거쳐 완성됩니다.
```

별도 고지 문구:

```text
제작 안정성을 위해 레이아웃 자유 편집, 코드 수정, 타 사이트 복제 방식의 제작은 제공하지 않습니다.
```

더 짧은 버전이 필요하면 다음을 쓴다.

```text
샘플은 제작 방향을 정하기 위한 참고안입니다.
최종 홈페이지는 학원 자료 확인과 묵산 검수 후 완성됩니다.
```

상담 중 구두 설명에 가까운 문구는 다음을 쓴다.

```text
샘플 중 우리 학원과 가장 가까운 분위기를 골라주시면, 묵산이 그 방향을 참고해 학원 자료와 운영 방식에 맞게 제작합니다.
샘플을 그대로 복사하는 방식은 아니며, 고객님이 직접 편집하는 제작툴도 아닙니다.
```

## 3. 공통 카드 구조

각 샘플 카드는 다음 구조를 가진다.

| 항목 | 문안 기준 |
|---|---|
| 카드 제목 | 고객이 쉽게 이해하는 한글명 |
| 방향 코드 | 내부 식별용 template/direction 후보 |
| 짧은 설명 | 1~2문장 |
| 잘 맞는 학원 | 고객이 자기 학원과 비교할 기준 |
| 잘 맞지 않는 경우 | 기대 불일치 방지 |
| 필요한 자료 | 제작 전 고객이 준비해야 할 정보 |
| 조정 가능 | 고객 권리로 허용할 선호 |
| 조정 불가 | 자유형 빌더 오해 차단 |
| 선택 전 고지 | 자동 적용/즉시 게시가 아님을 명시 |

버튼 문구는 당분간 다음 중 하나만 쓴다.

```text
이 방향 상담하기
이 방향으로 검토 요청
이 방향 메모하기
```

사용하지 않을 버튼 문구:

```text
바로 적용
이 템플릿으로 만들기
즉시 생성
내가 편집하기
사이트 복제하기
```

## 4. 카드 A: 신뢰형 기본

| 항목 | 문안 |
|---|---|
| 카드 제목 | 신뢰형 기본 |
| 방향 코드 | `trust-basic-v1` |
| 태그 | 안정감, 기본형, 중소형 학원 |
| 짧은 설명 | 학원의 수업 흐름과 관리 방식을 차분하게 보여주는 방향입니다. 처음 홈페이지를 준비하거나 자료가 많지 않은 학원도 부담 없이 검토할 수 있습니다. |
| 잘 맞는 학원 | 국어, 영어, 보습, 사회 등 단과 학원. 수업 철학, 관리 방식, 커리큘럼을 차분하게 설명하고 싶은 학원. |
| 잘 맞지 않는 경우 | 대형 브랜드 이미지, 입시 실적 중심 홍보, 화려한 시설 사진을 첫 화면에서 강하게 보여주고 싶은 경우. |
| 필요한 자료 | 학원명, 로고 또는 텍스트 로고 승인, 대표 문구, 대상 학년/과목, 핵심 강점 3개, 강사진 소개, 커리큘럼, 공지, 오시는 길. |
| 조정 가능 | 대표 색상 선호, 대표 이미지 방향, 강점 순서, 상담 문구 톤, 강사진 소개 방식. |
| 조정 불가 | 레이아웃 무제한 변경, 섹션 자유 조립, 코드/CSS 수정, 외부 사이트 복제. |

카드 하단 고지:

```text
기본 방향 선택 후에도 학원 자료와 묵산 검수에 따라 세부 구성은 조정될 수 있습니다.
```

상담용 한 줄 설명:

```text
첫 홈페이지 제작이나 자료가 많지 않은 학원에는 이 방향이 가장 안전합니다.
```

## 5. 카드 B: 내신/입시 관리형

| 항목 | 문안 |
|---|---|
| 카드 제목 | 내신/입시 관리형 |
| 방향 코드 | `exam-focus-v1` |
| 태그 | 시험 대비, 학년별 과정, 관리 체계 |
| 짧은 설명 | 시험 대비 과정, 학년별 커리큘럼, 오답 관리 흐름을 또렷하게 보여주는 방향입니다. 합격 실적이나 성과 표현은 확인 가능한 자료가 있을 때만 반영합니다. |
| 잘 맞는 학원 | 고등 내신, 수능, 중3-고등 전환, 학교별 시험 대비, 기출/오답 관리 체계가 있는 학원. |
| 잘 맞지 않는 경우 | 입시 실적 근거가 없는데 결과 중심 홍보를 원하는 경우. 초등 저학년, 독서, 기초 문해력 중심 학원. |
| 필요한 자료 | 대상 학년, 내신/수능 대비 방식, 학년별 커리큘럼, 자료 관리 방식, 과제/오답 피드백 루틴, 확인 가능한 실적 자료. |
| 조정 가능 | 내신 중심 또는 수능 중심 강조 비율, 학년별 과정 노출 순서, 시험 대비 공지 강조, 성과보다 관리 시스템을 앞세우는 구성. |
| 조정 불가 | 합격 보장, 등급 상승 보장, 근거 없는 적중률, 허위 후기, 실제 성적표나 학생 개인정보 노출. |

카드 하단 고지:

```text
실적과 성과 표현은 학원이 제공한 확인 자료가 있을 때만 반영합니다.
자료가 부족하면 관리 체계 중심으로 제작합니다.
```

상담용 한 줄 설명:

```text
시험 대비 체계가 분명한 학원에 적합하지만, 실적 표현은 검수 기준이 가장 엄격합니다.
```

## 6. 카드 C: 소수정예 관리형

| 항목 | 문안 |
|---|---|
| 카드 제목 | 소수정예 관리형 |
| 방향 코드 | `small-group-v1` |
| 태그 | 작은 반, 밀착 피드백, 학부모 소통 |
| 짧은 설명 | 작은 반, 과제 점검, 학부모 소통처럼 가까운 관리 방식을 보여주는 방향입니다. 실제 관리 루틴이 정리되어 있을수록 학원의 장점이 선명하게 드러납니다. |
| 잘 맞는 학원 | 반별 정원이 작고 과제, 오답, 학습 습관, 상담 기록을 꾸준히 관리하는 학원. |
| 잘 맞지 않는 경우 | 대형 입시 브랜드처럼 보이기를 원하는 경우. 실제 관리 방식이 정리되어 있지 않은데 밀착 관리를 주장하려는 경우. |
| 필요한 자료 | 반별 정원 기준, 과제 점검 방식, 오답/피드백 방식, 학부모 소통 방식, 학생 변화 관찰 기준, 강사진 역할, 커리큘럼. |
| 조정 가능 | 따뜻한 톤 또는 단정한 톤, 학부모 소통 강조 정도, 과제/오답/수업 후 점검 우선순위, 강사 이름 노출 여부. |
| 조정 불가 | 운영 현실을 넘는 완벽 관리 표현, 확인되지 않은 학생 변화 사례, 학부모 후기 창작, 프리미엄 이미지만 앞세우는 구성. |

카드 하단 고지:

```text
관리형 문구는 실제 운영 루틴을 기준으로 작성합니다.
구체적인 관리 방식이 없으면 감성 표현보다 기본형 구성을 권장합니다.
```

상담용 한 줄 설명:

```text
작은 반과 꾸준한 피드백이 실제 강점인 학원에 적합합니다.
```

## 7. 선택 전 고지

고객이 샘플 방향을 선택하기 전에 다음 고지를 보여준다.

```text
샘플 방향 선택은 홈페이지 제작을 위한 선호 확인 단계입니다.
선택하신 방향은 학원 자료, 사진, 검수 기준, 운영 가능 범위에 따라 조정될 수 있습니다.
묵산은 고객별 별도 코드베이스, 자유형 레이아웃 편집, CSS/코드 직접 수정, 타 사이트 복제 제작을 제공하지 않습니다.
```

간단한 확인 문구:

```text
샘플 방향은 최종 디자인 확정이 아니며, 묵산 검수 후 제작된다는 점을 확인했습니다.
```

이 확인 문구는 향후 선택 제출 UI가 생기더라도 고객 승인 체크박스가 아니라 기대 관리 확인에 가깝다. 즉시 게시 승인으로 쓰면 안 된다.

## 8. 선택 후 안내

선택 후 고객에게 보여줄 문구는 다음 방향이어야 한다.

```text
선호 방향이 접수되었습니다.
묵산은 제출하신 학원 자료와 선택 방향을 함께 검토한 뒤 제작 가능 범위와 필요한 추가 자료를 안내합니다.
최종 게시 전에는 고객 확인 절차를 거칩니다.
```

상담자가 내부 메모로 남길 때는 다음 형식을 쓴다.

```text
고객 선호 방향: 신뢰형 기본 / 내신·입시 관리형 / 소수정예 관리형
선택 이유:
필요한 추가 자료:
기대 관리 필요 여부:
검수 주의사항:
```

## 9. FAQ 문구

고객 질문에 대한 답변 기준은 다음과 같다.

| 질문 | 답변 |
|---|---|
| 샘플을 고르면 그대로 만들어지나요? | 샘플은 제작 방향을 정하기 위한 참고 자료입니다. 학원 자료와 검수 기준에 따라 세부 구성은 조정되며, 최종 게시 전 고객 확인 절차를 거칩니다. |
| 샘플 A의 메인과 샘플 B의 섹션을 섞을 수 있나요? | 원하시는 강조점은 참고할 수 있습니다. 다만 섹션을 자유롭게 조립하는 방식은 아니며, 묵산이 운영 가능한 표준 구조 안에서 제작합니다. |
| 우리 학원 색으로 바꿀 수 있나요? | 대표 색상 선호는 받을 수 있습니다. 다만 가독성과 디자인 토큰 기준에 맞게 묵산이 조정합니다. |
| 다른 학원 사이트와 똑같이 만들 수 있나요? | 참고하고 싶은 분위기는 들을 수 있습니다. 다만 타 사이트를 복제하지는 않으며, 묵산의 표준 제작 방식 안에서 새로 구성합니다. |
| 실적과 후기를 넣을 수 있나요? | 확인 가능한 실제 자료가 있을 때만 가능합니다. 허위 실적, 허위 후기, 확인되지 않은 경력은 사용하지 않습니다. |
| 선택 후 바로 게시되나요? | 아닙니다. 선택은 제작 방향을 정하기 위한 단계이며, 묵산 제작과 내부 검수, 고객 확인을 거친 뒤 게시됩니다. |

## 10. 문구 NO-GO

다음 문구는 샘플 갤러리나 상담 자료에서 사용하지 않는다.

- "원하는 대로 자유롭게 제작"
- "샘플을 고르면 즉시 자동 완성"
- "고객이 직접 편집 가능"
- "원하는 사이트 그대로 복제"
- "합격/성적 상승 보장"
- "후기와 실적은 AI가 알아서 작성"
- "디자인 제한 없음"
- "검수 없이 바로 게시"

대체 문구:

| 금지 방향 | 대체 문구 |
|---|---|
| 자유 제작 | 표준 템플릿 안에서 학원 자료에 맞게 제작 |
| 즉시 생성 | 묵산 내부 제작과 검수 후 완성 |
| 사이트 복제 | 참고 분위기를 반영하되 새로 제작 |
| 성과 보장 | 확인 가능한 자료를 기준으로 표현 |
| 고객 직접 편집 | 고객은 선호와 자료를 제공하고 묵산이 제작 |

## 11. 이번 문서의 결론

샘플 카드 문안은 고객의 선택권을 인정하되 선택권의 범위를 좁고 명확하게 설명해야 한다.

좋은 문구의 기준:

- 고객이 자신의 학원에 맞는 방향을 고를 수 있다.
- 자유형 빌더로 오해하지 않는다.
- 허위 실적과 무한 맞춤 기대를 만들지 않는다.
- 묵산 내부 제작과 검수 절차가 분명하게 보인다.

이 문서는 갤러리 화면 구현의 입력 자료일 뿐, 아직 구현 착수 결정은 아니다.
