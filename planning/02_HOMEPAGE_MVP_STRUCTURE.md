# 홈페이지 MVP 페이지 구조

Status: Draft  
Project: homepage  
Last Updated: 2026-06-24

---

## 1. 목적

이 문서는 묵산 홈페이지 프로젝트의 첫 번째 실사용 버전에서 제공할 기본 페이지와 정보 구조를 정의한다.

MVP는 많은 기능을 넣는 것이 아니라, 학부모가 신뢰하고 상담 신청할 수 있는 완성도 높은 기본 홈페이지를 제공하는 것을 목표로 한다.

---

## 2. MVP 기본 페이지

초기 홈페이지는 다음 페이지를 기본으로 한다.

| 페이지 | 목적 |
|---|---|
| 메인 | 학원의 첫인상과 핵심 강점 전달 |
| 학원 소개 | 학원의 철학, 운영 방식, 신뢰 요소 전달 |
| 강사진 | 선생님 소개와 전문성 전달 |
| 커리큘럼 | 대상 학년, 과목, 수업 방식 안내 |
| 시간표 또는 수업 안내 | 실제 수업 운영 정보 제공 |
| 공지사항 | 최근 소식과 운영감 전달 |
| 상담 문의 | 입학 상담 접수 |
| 오시는 길 | 위치, 교통, 연락처 안내 |
| 수강료 결제 진입 | 재원 학부모의 결제 접근 |
| 관리자 진입 | 원장·강사의 관리 시스템 접근 |

---

## 3. 메인 페이지

메인 페이지는 홈페이지의 첫인상이다.

반드시 포함해야 할 정보:

- 학원명
- 대표 문구
- 대표 과목
- 대상 학년
- 핵심 강점 3개
- 상담 신청 버튼
- 전화 또는 연락 버튼
- 위치 요약
- 최근 공지 또는 주요 안내

메인 페이지의 목적은 모든 정보를 길게 설명하는 것이 아니라, 학부모가 계속 둘러볼 이유를 만드는 것이다.

---

## 4. 학원 소개 페이지

학원 소개 페이지는 학원의 신뢰를 만드는 페이지다.

포함 정보:

- 학원 운영 철학
- 수업 방향
- 관리 방식
- 대상 학생
- 학원의 차별점
- 시설 또는 수업 분위기 사진

과장 광고보다 실제 운영감과 신뢰감을 우선한다.

---

## 5. 강사진 페이지

강사진 페이지는 학부모가 선생님을 신뢰할 수 있게 만드는 페이지다.

포함 정보:

- 강사명
- 담당 과목
- 담당 학년
- 수업 특징
- 경력
- 사진

금지 사항:

- 확인되지 않은 경력
- 허위 수상
- AI가 만들어낸 실적
- 가짜 후기

---

## 6. 커리큘럼 페이지

커리큘럼 페이지는 학부모가 “우리 아이에게 맞는 수업이 있는가”를 판단하는 페이지다.

포함 정보:

- 과목
- 대상 학년
- 수업 목표
- 수업 방식
- 교재 또는 자료 활용 방식
- 내신/수능/기초/심화 구분
- 반 구성

---

## 7. 시간표 또는 수업 안내 페이지

초기에는 복잡한 실시간 시간표 시스템을 만들지 않는다.

기본 정보는 다음 정도로 제한한다.

- 요일
- 시간대
- 대상 학년
- 과목
- 반 이름
- 상담 필요 여부

장기적으로는 academy의 반·시간표 데이터와 연결할 수 있어야 한다.

---

## 8. 공지사항 페이지

공지사항은 학원이 실제로 운영되고 있다는 느낌을 준다.

초기 공지 기능은 단순하게 시작한다.

포함 정보:

- 제목
- 작성일
- 내용
- 고정 공지 여부
- 공개 여부

장기적으로는 academy의 공지·문자·알림과 연결한다.

---

## 9. 상담 문의 페이지

상담 문의는 MVP의 핵심 전환 기능이다.

필수 입력:

- 보호자 이름
- 연락처
- 학생 학년
- 관심 과목
- 문의 내용
- 개인정보 수집 동의

상담 문의는 장기적으로 academy 상담함으로 접수되어야 한다.

초기에는 저장 또는 이메일 알림 방식으로 시작할 수 있다.

---

## 10. 오시는 길 페이지

포함 정보:

- 주소
- 지도
- 전화번호
- 운영 시간
- 교통 안내
- 주차 안내

---

## 11. 수강료 결제 진입

초기 홈페이지는 결제를 직접 처리하지 않는다.

수강료 결제는 academy의 청구·결제 구조와 연결한다.

초기에는 다음 형태 중 하나로 둔다.

- 결제 준비 중 안내
- 재원생 전용 결제 링크
- 관리자 발송 결제 링크로 이동

민감 카드정보는 홈페이지 시스템이 직접 처리하지 않는다.

---

## 12. 관리자 진입

원장 또는 강사가 묵산 관리자 시스템으로 들어갈 수 있는 진입 버튼을 둔다.

연결 대상:

- 학원관리 프로그램
- 시험지 생성관리 프로그램
- 홈페이지 관리 화면

초기에는 단순 링크 또는 로그인 예정 안내로 시작할 수 있다.

---

## 13. 모바일 기준

홈페이지는 모바일 우선으로 설계한다.

학부모는 PC보다 스마트폰으로 학원을 검색하고 상담할 가능성이 높다.

모바일에서 다음이 쉬워야 한다.

- 전화하기
- 상담 신청
- 지도 보기
- 커리큘럼 확인
- 공지 확인
- 결제 링크 접근

---

## 14. MVP에서 하지 않는 것

초기 MVP에서는 다음을 하지 않는다.

- 자유형 페이지 편집
- 블로그형 고급 CMS
- 실시간 채팅
- 네이티브 앱
- 학생별 개인정보 화면
- 완전한 학부모 포털
- 자체 PG 결제 처리
- 고객별 무제한 디자인 수정

---

## 15. 한 줄 정의

MVP 홈페이지는 학부모가 신뢰하고 상담 신청할 수 있는 고품질 기본 홈페이지다.
