# 로컬 시연 시각 판정 기록

Status: Draft
Project: homepage
Last Updated: 2026-07-01

---

## 1. 목적

이 문서는 묵산이 로컬 시연 URL로 공개 샘플 홈페이지를 직접 확인한 뒤 내린 시각 판정 결과를 기록한다.

이번 판정은 홈페이지 프로젝트의 MVP 공개 샘플 화면이 다음 단계로 넘어갈 수 있는지 확인하기 위한 내부 판정이다.

---

## 2. 판정 정보

| 항목 | 값 |
|---|---|
| 판정일 | 2026-07-01 |
| 판정자 | 묵산 |
| 판정 결과 | `GO` |
| 판정 입력 | `go. 다음` |
| 확인 URL | `http://127.0.0.1:5176/h/sample-korean-academy` |
| URL 성격 | 임시 로컬 검증 URL |
| 공식 로컬 시연 포트 | `5175` |
| 현재 `5175` 상태 | 다른 workspace 점유로 HOLD |

이번 확인 URL은 공식 로컬 시연 URL이 아니라 임시 검증 URL이다.

`5175` 포트는 공식 기준으로 유지한다. 다만 현재 PC에서는 `academy/frontend`가 `5175`를 점유하고 있어, 이번 판정은 `5176` 임시 URL에서 진행했다.

---

## 3. 판정 대상

묵산이 GO로 판정한 대상:

- 공개 샘플 홈페이지 첫인상
- 학부모가 볼 때의 신뢰감
- 모바일/데스크톱 기본 레이아웃
- 상담 신청 버튼과 상담 폼 흐름
- 국어학원 샘플에 어울리는 화면 톤
- 과장 광고처럼 보이지 않는 문구 방향

---

## 4. 판정 제외 항목

이번 GO에 포함되지 않는 항목:

- 실제 고객 발송 승인
- 외부 공유 URL 승인
- 운영 배포 승인
- 커스텀 도메인 승인
- 실제 고객 자료 사용 승인
- `/internal` 내부 제작 화면 외부 공개 승인
- 고객별 홈페이지 제작 완료 승인

즉, 이번 GO는 **MVP 공개 샘플 홈페이지의 시각 방향 승인**이다.

현재 상태:

```text
공개 샘플 홈페이지 시각 방향: GO
공식 로컬 포트 5175: HOLD until conflict resolved
임시 검증 URL 5176: usable for this session only
외부 공유 URL: NOT READY
운영 배포: NOT APPROVED
```
